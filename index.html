Como usar o cronômetro

Este cronômetro permite medir o tempo com precisão e pode ser utilizado para diversas finalidades, como calcular a duração de uma atividade, medir o tempo de apneia ou simplesmente cronometrar eventos. Passos para utilizar: Iniciar o cronômetro Clique no botão "Iniciar" (ou acione o código correspondente). O tempo começará a ser contado automaticamente, atualizando a tela. Parar o cronômetro Clique no botão "Parar" para interromper a contagem. O cronômetro pausará no momento exato em que for interrompido. Resetar o cronômetro Se desejar reiniciar a contagem do zero, clique em "Resetar". Todos os valores voltarão para 00:00.00.
